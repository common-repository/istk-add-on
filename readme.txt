This plugin adds features for theme "ISTK Portfolio".

== Description ==

This plugin adds features for theme "ISTK Portfolio". ISTK Portfolio is the theme for creating a portfolio site to get commissions for illustrators, artists, designers and more.

Plese check [ISTK Portfolio](https://wordpress.org/themes/istk-portfolio/).

You may use this plugin without ISTK Portfolio theme, but we cannot guarantee that pages or parts will be displayed correctly.

= Features =
* Add the post type and template file for portfolio page.
* Add the category and tag for portfolio.
* You can set the image for each portfolio category.
* Add the post type for news.
* Add shortcodes to display portfolio category panel and portfolio tag cloud.
* And add some more settings.


== Installation ==
1. Navigate to Plugins > Add New page in your admin page then search "**ISTK Add-On**". Click the "Install Now" link.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Settings > ISTK Portfolio Settings then change settings.
